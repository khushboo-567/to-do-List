# To-Do List App

A clean and interactive To-Do List web application built using HTML, CSS, and JavaScript. The application helps users efficiently manage their daily tasks with an intuitive interface and persistent local storage.

# Features

### ➕ Add Tasks

Users can quickly add new tasks by typing into the input field and clicking the **Add** button. Empty or whitespace-only tasks are not allowed, ensuring only valid tasks are added to the list.

### ✅ Mark Tasks as Completed

Clicking on a task marks it as completed by toggling its status. Completed tasks are visually distinguished, making it easy to identify finished work.

### ✏️ Edit Existing Tasks

Users can modify the text of any task by clicking the **Edit** button. The task becomes editable, and changes can be saved by pressing **Enter** or by clicking outside the input field.

### ❌ Delete Tasks

Tasks that are no longer needed can be permanently removed by clicking the **× (Delete)** button, helping keep the task list organized.

### 📊 Task Counter

The application displays real-time task statistics:

* **Total:** Total number of tasks created.
* **Completed:** Number of tasks marked as completed.
* **Remaining:** Number of tasks that are yet to be completed.

The counters update automatically whenever a task is added, completed, edited, or deleted.

### 🔍 Task Filtering

Users can filter tasks based on their status:

* **All:** Displays every task.
* **Completed:** Displays only completed tasks.
* **Remaining:** Displays only pending tasks.

This makes it easier to focus on specific tasks without scrolling through the entire list.

### 💾 Local Storage Support

All tasks are automatically saved in the browser's **Local Storage**. This ensures that the task list remains available even after refreshing the page or reopening the browser, without requiring a database or user account.

### 📱 Clean and Responsive Interface

The application features a simple, user-friendly interface designed with HTML and CSS. The layout is responsive and remains easy to use across different screen sizes.
